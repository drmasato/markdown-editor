# Markdown Editor

シンプルなリアルタイムMarkdownエディタです。インストール不要、ブラウザで動作します。

## 起動方法

`index.html` をダブルクリックするだけで使えます。

---

## 画面構成

```
┌─────────────────────────────────────────────────────┐
│  ツールバー（太字・保存など）                          │
├──────────────────────┬──────────────────────────────┤
│                      │                              │
│   Editor（左）        │   Preview（右）               │
│   Markdownを入力       │   リアルタイムで表示           │
│                      │                              │
├──────────────────────┴──────────────────────────────┤
│  文字数 / 行数 / 単語数                               │
└─────────────────────────────────────────────────────┘
```

左右のパネルは中央の仕切りをドラッグしてサイズ変更できます。

## 使い方

### 基本的な書き方

| 書き方 | 表示 |
|--------|------|
| `# 見出し1` | 大見出し |
| `## 見出し2` | 中見出し |
| `**太字**` | **太字** |
| `*斜体*` | *斜体* |
| `` `コード` `` | インラインコード |
| `- 項目` | 箇条書き |
| `1. 項目` | 番号付きリスト |
| `> 引用` | 引用ブロック |
| `---` | 水平線 |

## ファイルの読み込み

2通りの方法でMarkdownファイルを開けます。

**1. ボタンから開く**
ツールバーの「ファイルを開く」をクリックしてファイルを選択します。

**2. ドラッグ＆ドロップ**
`.md` / `.txt` / `.markdown` ファイルをエディタ左側にドロップするだけで読み込めます。

読み込んだファイル名はステータスバーとエディタラベルに表示されます。「保存」ボタンは元のファイル名で保存します。

---

## ツールバー機能

| ボタン | 機能 |
|--------|------|
| **B** | 選択テキストを太字にする |
| *I* | 選択テキストを斜体にする |
| Code | 選択テキストをインラインコードにする |
| Block | コードブロックを挿入する |
| H2 | 行頭に `## ` を追加する |
| List | 行頭に `- ` を追加する |
| ファイルを開く | .md / .txt ファイルを読み込む |
| HTML コピー | プレビューのHTMLをクリップボードにコピー |
| 保存 | Markdownファイル（.md）としてダウンロード |
